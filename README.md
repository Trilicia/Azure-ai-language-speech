
# **Azure Speech Studio e Language Studio**
**Descrição**

Este repositório foi desenvolvido como parte do laboratório prático utilizando as ferramentas Azure Speech Studio e Azure Language Studio, com foco em inteligência artificial aplicada à fala e linguagem natural.

O objetivo do desafio foi explorar recursos de:

- Reconhecimento de fala;
- Conversão de texto em voz;
- Análise de sentimentos;
- Processamento de linguagem natural;
- Compreensão de texto utilizando IA.

# **Objetivos de Aprendizagem**
- Aplicar conceitos de IA em ambiente prático;
- Utilizar ferramentas da Microsoft Azure;
- Documentar processos técnicos;
- Organizar documentação técnica utilizando GitHub.

# **Ferramentas Utilizadas**
- Azure Speech Studio
- Azure Language Studio
- GitHub
- Markdown

# **Speech Studio**
## **O que é**

O Azure Speech Studio é uma plataforma da Microsoft voltada para soluções de voz utilizando inteligência artificial.

Com ela é possível:

- Converter fala em texto;
- Converter texto em fala;
- Traduzir áudio;
- Reconhecer comandos de voz.

## **Atividades Realizadas**

**Conversão de fala para texto**

Foi realizado um teste utilizando reconhecimento automático de fala, onde a plataforma identificou o áudio e transformou em texto com alta precisão.

**Conversão de texto em fala**

Foi utilizado o recurso de síntese de voz para gerar áudio artificial a partir de textos digitados.

# **Language Studio**
## **O que é**

O Azure Language Studio permite trabalhar com processamento de linguagem natural (NLP), utilizando inteligência artificial para interpretação de textos.

## **Atividades Realizadas**

**Análise de Sentimentos**

Foi realizado um teste para identificar sentimentos em frases positivas, negativas e neutras.

**Extração de Entidades**

A ferramenta identificou automaticamente:

- nomes;
- locais;
- datas;
- organizações.

**Detecção de Idioma**

Foi possível identificar automaticamente o idioma de diferentes frases.

# **Insights e Aprendizados**

Durante o laboratório foi possível compreender como serviços cognitivos da Azure podem ser utilizados em aplicações reais.

**Principais aprendizados**
- A IA de voz possui aplicações em assistentes virtuais e automação;
- O reconhecimento de fala apresentou boa precisão;
- A análise de sentimentos pode auxiliar empresas em atendimento ao cliente;
- O Language Studio facilita tarefas complexas de NLP sem necessidade de criar modelos do zero.
